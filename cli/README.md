# itwin-pcf cli

The pcf toolkit provides the pcf command-line interface that can be used to work with pcf-powered iTwin.js Connector Apps. 

Command                           | Description
----------------------------------|-------------------------------------------------------------------------------------
[`pcf init [connector-name]`](#pcf-init)  | Start a new pcf connector project.
