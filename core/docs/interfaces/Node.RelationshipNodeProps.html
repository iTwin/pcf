Interface RelationshipNodeProps

All Node have a unique identifier. Do not confuse this identifier with those of IRInstance, which are found in the source data. Node identifiers are used to locate the ECInstanceId of the corresponding element in the iModel when the node corresponds to exactly one element in the iModel. For example, ModelNode, which can't have a DMO.
